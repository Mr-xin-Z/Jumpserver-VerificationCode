# Jumpserver-VerificationCode

### 介绍
本项目为[jumpserver](https://github.com/jumpserver/jumpserver)添加了企业微信验证码功能   
### 使用方法
1. 3个脚本放置与jumpserver根目录
2. 修改 **/etc/passwd** 中的**init.sh**修改为**NewInit.sh**
